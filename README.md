# Vuejs Todo App  ⚡️
![](demo.gif)
## A clean, beautiful and responsive To do App for Vue.js Developers!

## Sections 
✔️ Show Task List\
✔️ Add the Task\
✔️ Delete the Task \
✔️ Check and uncheck Task\

## Technologies used 🛠️

- [Vue Js](https://vuejs.org/)
- [SAAS]

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```
